Metric Collector

Simple, self-hosted metric aggregator and dashboard

GitHub | Documentation | License

A lightweight metric collection and visualization tool. No Prometheus, no complex setup. Just send metrics via HTTP POST and view them in a dashboard.

Quick Start

docker-compose up -d
# Dashboard: http://localhost:3000

Features

  • Simple POST API for metric collection
  • Multi-tenancy with App IDs
  • Role-based API keys (read/write/admin)
  • Auto-generated dashboards
  • Threshold alerts with SNS
  • SQLite, PostgreSQL, or MySQL support
  • Data retention policies
  • CloudWatch integration

Usage

POST /api/metrics
{
  "metric": "cpu_usage",
  "value": 75.5,
  "app_id": "myapp"
}

Requirements

  • Node.js 18+
  • SQLite (included) or PostgreSQL/MySQL
  • Optional: Docker for easy deployment

Use Cases

  • Application performance monitoring
  • Business metrics tracking
  • IoT sensor data collection
  • Simple observability for small projects
  • Lightweight alternative to Prometheus
